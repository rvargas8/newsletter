# Security & Privacy Policy

## Overview
The RAMS Roundup Newsletter website is designed with security and privacy in mind. We protect student and parent information while providing a secure educational resource.

## Security Measures

### Data Protection
- We do not collect or store personal student information on this website
- No user accounts or login credentials are required
- All email links use standard mailto: protocol

### Website Security
- **Security Headers**: The site implements security headers to prevent common attacks
- **External Links**: All external links use `rel="noopener noreferrer"` to prevent tabnabbing
- **Content Security**: Meta tags prevent XSS and clickjacking attacks
- **HTTPS**: Site is served over encrypted connections

### Privacy
- **No Tracking**: No analytics or tracking cookies are used
- **No Personal Data Collection**: The site is purely informational
- **Public Information**: Only public educational content is displayed
